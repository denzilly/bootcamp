## Virtual Environments

  To create a virtual environment:

  ~~~py

  conda create --name envname envpackage

  activate envname

  deactivate
  ~~~

  To create an environment using a different version of python:

  ~~~Py

  conda create --name envname python=version package

  ~~~~
